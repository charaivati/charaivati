## Reporting a Vulnerability

If you discover a security vulnerability in **Charaivati**, please report it responsibly.

**Contact:**

When reporting, please include:
- A clear description of the issue.
- Steps to reproduce (if applicable).
- Potential impact or affected components.

We kindly ask that you **do not publicly disclose** the vulnerability until we have confirmed and resolved it.

Once submitted, our team will:
1. Acknowledge receipt within **48 hours**.  
2. Begin an internal investigation.  
3. Provide progress updates (if needed).  
4. Release a patch or mitigation steps.  
5. Credit you appropriately (if you wish).
